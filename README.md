# Automatización de Matrices de Transición

Generación automatizada de matrices analíticas y reportes segmentados utilizando Python, BigQuery y Google Sheets.

## Descripción

Proyecto orientado al procesamiento y segmentación de información mediante consultas SQL y generación automática de matrices de transición y reportes.

Incluye extracción, transformación y publicación automatizada de resultados para reducir tareas operativas manuales.

## Funcionalidades

* Extracción automatizada desde BigQuery
* Procesamiento y segmentación de información
* Construcción de matrices analíticas
* Generación automática de reportes
* Exportación a Google Sheets

## Tecnologías

* Python
* Pandas
* BigQuery
* SQL
* Google Sheets
